### `실행방법`

## 로그인/회원가입

`/` 로그인  
`/sign` '회원가입

- 회원가입 경우 이메일 조건: @ 포함등 xxxxxxx@xxx.xx 처럼 이메일 형식을 갖춰야됩니다.
  비밀번호 조건: 8자 이상,영문,숫자,특수문자으로 기입해야합니다.
- 로그인은 이메일과 비밀번호가 데이터가 없으면 로그인 버튼을 눌러도 활성 못하게 지정했습니다.
